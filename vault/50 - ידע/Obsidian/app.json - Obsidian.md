# app.json (Obsidian)

**נתיב:** `vault/.obsidian/app.json`
**בעלים:** [[ראובן]] (מנוהל אוטומטית)
**סוג:** קובץ JSON עם הגדרות אפליקציה כלליות.

## מה הקובץ עושה

מכיל הגדרות כלליות של אפליקציית Obsidian: רוחב טאבים, גודל גופן בסיסי, התנהגות חלון, אינטגרציה עם המערכת.

## תוכן נוכחי

`{}` — ריק לחלוטין. כלומר Obsidian משתמש בכל ברירות המחדל. כשראובן ישנה הגדרה בממשק (Settings → Editor / General), הקובץ יתעדכן.

## למי הוא משויך

[[ראובן]] בעלים, מתעדכן אוטומטית.
