# core-plugins.json (Obsidian)

**נתיב:** `vault/.obsidian/core-plugins.json`
**בעלים:** [[ראובן]] (מנוהל אוטומטית)
**סוג:** קובץ JSON עם flags של פלאגינים בסיסיים.

## מה הקובץ עושה

מציין אילו מהפלאגינים המובנים של Obsidian פעילים.

## מה מופעל

| פלאגין | סטטוס | תיאור קצר |
| --- | --- | --- |
| `file-explorer` | ✅ | סייר הקבצים בצד שמאל |
| `global-search` | ✅ | חיפוש בכל הוולט |
| `switcher` | ✅ | מעבר מהיר בין קבצים (Ctrl+O) |
| `graph` | ✅ | תצוגת הגרף — ראה [[graph.json - Obsidian]] |
| `backlink` | ✅ | מציג מי מקשר לפתק הנוכחי |
| `canvas` | ✅ | קנבסים לתכנון ויזואלי |
| `outgoing-link` | ✅ | מציג קישורים יוצאים |
| `tag-pane` | ✅ | חלונית תגיות |
| `properties` | ✅ | תצוגת frontmatter |
| `page-preview` | ✅ | תצוגה מקדימה בריחוף |
| `daily-notes` | ✅ | פתקים יומיים (טרם הופעלו בפועל) |
| `templates` | ✅ | טמפלטים |
| `note-composer` | ✅ | עריכת מבנה פתק |
| `command-palette` | ✅ | פלטת פקודות (Ctrl+P) |
| `editor-status` | ✅ | סטטוס בעורך |
| `bookmarks` | ✅ | מועדפים |
| `outline` | ✅ | חלונית outline |
| `word-count` | ✅ | ספירת מילים |
| `file-recovery` | ✅ | גיבוי גרסאות |
| `sync` | ✅ | Obsidian Sync (אופציונלי, דורש מנוי) |
| `bases` | ✅ | בסיסי נתונים |

## מה כבוי

`footnotes`, `slash-command`, `markdown-importer`, `zk-prefixer`, `random-note`, `slides`, `audio-recorder`, `workspaces`, `publish`, `webviewer`.

## למי הוא משויך

[[ראובן]] בעלים. הוא מחליט אילו פלאגינים להפעיל/לכבות דרך Settings → Core plugins.
