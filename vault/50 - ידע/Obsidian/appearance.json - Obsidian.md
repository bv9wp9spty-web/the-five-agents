# appearance.json (Obsidian)

**נתיב:** `vault/.obsidian/appearance.json`
**בעלים:** [[ראובן]] (מנוהל אוטומטית)
**סוג:** קובץ JSON עם הגדרות מראה.

## מה הקובץ עושה

מכיל הגדרות חזותיות: תמה (אור/חושך), גודל גופן בסיסי, צבע מבטא, תמות מותקנות.

## תוכן נוכחי

`{}` — ריק. ברירות מחדל של Obsidian: בדרך כלל תמה בהירה ופונט מערכת.

## למי הוא משויך

[[ראובן]] בעלים, מתעדכן כשהוא משנה תמה ב-Settings → Appearance.
