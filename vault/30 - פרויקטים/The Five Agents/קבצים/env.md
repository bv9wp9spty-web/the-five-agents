# .env

**נתיב:** `.env`
**בעלים:** [[ראובן]] (סודות פרטיים)
**סוג:** קובץ סביבה עם מפתחות API אמיתיים.

## מה הקובץ עושה

מכיל את המפתחות האמיתיים של ה-APIs שהצוות צורך: Anthropic, OpenAI, Tavily, Perplexity, וכן הגדרות כלליות (`DEFAULT_MODEL`, `LOG_LEVEL`).

נכון לעכשיו (2026-05-13) הערכים **ריקים** — הקובץ קיים כתשתית ועדיין לא מולא במפתחות אמיתיים.

## אבטחה

- **לא ב-Git.** נמצא ב-[[gitignore]] (`.env`).
- אסור להעלות, אסור לשלוח, אסור להציג.
- אם צריך לשתף — להעביר רק את [[env.example]] (התבנית הריקה).

## מבנה

זהה למבנה של [[env.example]] — אותם מפתחות, אבל עם ערכים אמיתיים במקום placeholders.

## למי הוא משויך

[[ראובן]] (CEO) מחזיק במפתחות. הם משמשים את כל הצוות:

- `ANTHROPIC_API_KEY` — כל הסוכנים (גישה ל-Claude).
- `OPENAI_API_KEY` — [[יובל]] (יצירת תמונות).
- `TAVILY_API_KEY` — [[חן]] (חיפוש).
- `PERPLEXITY_API_KEY` — [[חן]] (חיפוש).

## קבצים קשורים

- [[env.example]] — התבנית הציבורית.
- [[gitignore]] — הקובץ שמונע ממנו להיכנס ל-Git.
