# Output (תקייה)

**נתיב:** `Output/`
**בעלים:** [[יעל]] (כותבת).
**סטטוס:** **פעילה.** הוקמה ב-2026-05-13. כרגע ריקה (רק `.gitkeep`).

## מה התקייה עושה

מקום היציאה של תהליך הכתיבה. [[יעל]] שומרת כאן את התוצרים אחרי שכתוב המאמר שנשלף מ-[[Content]].

## מבנה התוצרים

לכל מאמר גלמי, [[יעל]] יוצרת **שני קבצים** באותו שם בסיס:

- `<name>.md` — גרסת Markdown של התוצר המשוכתב.
- `<name>.html` — גרסת HTML נקייה לקריאה (טמפלייט בסיסי, `dir="rtl"` אם בעברית, רוחב מקסימלי ~700px).

אין מבנה תיקיות פנימי כרגע — קבצים שטוחים.

## Flow

1. [[יעל]] מסיימת שכתוב.
2. כותבת `<name>.md` ו-`<name>.html` ל-`Output/`.
3. מחזירה ל-[[ראובן]] סיכום שכולל את שני הנתיבים.

## למי היא משויכת

- [[יעל]] — היחידה שכותבת לכאן.
- [[ראובן]] — צורך את התוצר ומציג אותו למשתמש.

## קבצים קשורים

- [[Content]] — הכניסה המקבילה.
- [[yael]] — תקיית העבודה של יעל.
- [[yael (סוכן)]] — קובץ ההגדרה של יעל, מפנה ל-`Output/` במפורש.
- [[CLAUDE.md - תיעוד]] — מציין את `Output/` כפלט של יעל.
