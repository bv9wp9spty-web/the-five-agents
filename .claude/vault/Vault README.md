# Vault README — הוולט של ראובן והצוות

ברוך הבא לוולט. זהו ה-Single Source of Truth לכל מה שקשור לפרויקט **The Five Agents** — צוות הסוכנים של ראובן ליצירת תוכן.

## איך הוולט בנוי

הוולט מאורגן לפי הנוהל של [[obsidian-vault-workflow]]:

- [[00 - מערכת]] — חוקי הוולט, MOCs, טמפלטים, וההגדרה של הסקיל הזה עצמו.
- [[30 - פרויקטים]] — פרויקטים פעילים. כרגע: **The Five Agents**.
- [[40 - סוכנים]] — דף לכל סוכן בצוות: ראובן (CEO), יעל, יובל, חן.
- [[50 - ידע]] — ידע תומך. כרגע: מפת ה-Superpowers שמותקנים בפרויקט.

## נקודות התחלה מומלצות

- [[MOC - הכל]] — מפת הניווט הראשית של הוולט.
- [[Project Hub]] — מרכז הפרויקט "The Five Agents".
- [[MOC - הצוות]] — סקירה של ארבעת חברי הצוות.

## עקרונות (תזכורת קצרה)

1. **פתק אחד = רעיון אחד**. במקום מסמך ענק, פתקים אטומיים מקושרים.
2. **קישורים עם `[[שם פתק]]`** במקום הכפלת תוכן.
3. **YAML frontmatter** בכל פתק משמעותי.
4. **ארכוב, לא מחיקה**. מעבירים לתקיית `90 - ארכיון/` אם נוצרת.
5. **עברית לכותרות, אנגלית קטנה לתגיות.**
