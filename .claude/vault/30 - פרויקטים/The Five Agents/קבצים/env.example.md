# .env.example

**נתיב:** `.env.example`
**בעלים:** [[ראובן]]
**סוג:** תבנית של קובץ הסביבה. ב-Git, ללא ערכים אמיתיים.

## מה הקובץ עושה

מציג בפני כל מי שמשכפל את הפרויקט אילו משתני סביבה נדרשים. הוא קובץ הוראות, לא קובץ סודות.

## מבנה ותוכן

חמש קבוצות מפתחות עם הערות בעברית שמסבירות לאיזה סוכן כל מפתח שייך:

| משתנה | תיאור | סוכן בעל הקובץ |
| --- | --- | --- |
| `ANTHROPIC_API_KEY` | Claude API לכל הסוכנים | כל הצוות |
| `OPENAI_API_KEY` | יצירת תמונות ומודלים נוספים | [[יובל]] |
| `TAVILY_API_KEY` | חיפוש ומחקר | [[חן]] |
| `PERPLEXITY_API_KEY` | חיפוש ומחקר | [[חן]] |
| `DEFAULT_MODEL` | מודל ברירת מחדל (כרגע `claude-opus-4-7`) | הגדרה כללית |
| `LOG_LEVEL` | רמת לוגים (כרגע `info`) | הגדרה כללית |

## למי הוא משויך

- [[ראובן]] — מתחזק את הקובץ ומעדכן כשמוסיפים מפתחות חדשים.
- [[יעל]], [[יובל]], [[חן]] — כל אחד מהם זקוק לסט מפתחות אחר. הקובץ הוא **המקור הרשמי** של ההחלטה איזה סוכן צורך איזה API.

## קבצים קשורים

- [[env]] — הקובץ האמיתי עם הערכים.
- [[gitignore]] — מציין שרק `.env` (לא `.env.example`) מוחרג מ-Git.
- [[יובל]], [[חן]] — מצוטטים מפורשות בהערות הקובץ.
